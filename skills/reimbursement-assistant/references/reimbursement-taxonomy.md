# Reimbursement Taxonomy

Use these categories as normalized internal values. Map them to company template labels through `template-map.json`.

| Category | Meaning |
| --- | --- |
| `intercity_transport` | Train, flight, long-distance bus, ship, intercity travel tickets |
| `local_transport` | Taxi, ride-hailing, metro, bus, airport transfer |
| `lodging` | Hotel, accommodation invoice, lodging payment |
| `meals` | Meals that are reimbursed by receipt, not per-day subsidy |
| `subsidy_meal` | Per-day meal subsidy |
| `subsidy_local_transport` | Per-day local transportation subsidy |
| `office` | Office supplies and small work expenses |
| `communication` | Phone, network, postage |
| `meeting` | Meeting fee and meeting-related materials |
| `entertainment` | Business reception, meals with guests |
| `procurement` | Purchases, materials, fixed assets |
| `payment_record` | Payment proof that is not an invoice |
| `approval_material` | Preapproval, trip request, meeting notice |
| `other` | Unknown reimbursable item |
| `non_reimbursable` | Private, refund, transfer, income, unrelated item |
